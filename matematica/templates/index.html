Ajuda Fórmula de Bhaskara

Seja a função f(x) = ax² + bx + c

Basca substibuir os valores de a,b e c na fórmula de Bhaskara:

Δ = b²-4*a*c

Ajuda Calcular X' e X''

Seja a função f(x) = ax² + bx + c

Para calcular x', basta substituir os valores de a, b e delta na seguinte equação:

x = (-b + square(Δ))/2*a


Para calcular x'', basta substituir os valores de a, b e delta na seguinte equação

x = (-b - square(Δ))/2*a

Ajuda Coordenada do Vértice da Parábola

Seja a função f(x) = ax² + bx + c

Para calcular a Coordena do vértices basta substituir os valores de a, b e delta:

Coordenada = (-b /2*a , -Δ/4*a)
